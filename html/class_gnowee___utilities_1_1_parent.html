geometry[MCNP_Geometry object] The geometry for running the MCNP radiation trasport code. Contains the surfaces, cells, and material information
GS[Gnowee Settings object] An object representing the settings for the optimization algorithm
mcnp[MCNP_Settings object] An object representing the settings for running the MCNP radiation trasport code. Contains the source, physics, and tally information.

iinteger (optional) Parent indexed location for initial LHC sampling purposes fitness float (optional) The assessed design fitness
