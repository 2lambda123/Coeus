Creates a MCNP surface object.

Currently can handle SO, (PX,PY,PZ), (CX,CY,CZ), RCC, RPP, and TRC surfaces and macrobodies. All others will throw an exception. Attributes not used are specified as -1. All atribute names follow those shown in Table 3.1 and Section 3.III.D in Volume II of the MCNP manual
